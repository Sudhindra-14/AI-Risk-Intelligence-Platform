# AI Risk Intelligence Platform

## About

This project is a simple system that analyzes user activity and predicts whether the behavior is normal or risky.

I built this to understand how real systems detect unusual patterns using machine learning.

## What it does

- Takes user activity data  
- Uses an ML model to detect anomalies  
- Classifies users as Normal or High Risk  

## Tech used

Python  
Pandas  
Scikit-learn  

## Use case

Can be used in fraud detection, security monitoring, and risk analysis systems.
## Output

- Classifies users into Normal or High Risk  
- Helps identify abnormal behavior patterns  

## Learning

- Learned how anomaly detection works  
- Understood how ML can be applied to real-world problems
